Instructions for Athlete Biographies

This document explains how to display athlete biographies automatically for the current athlete.

Install an HTML editing program

For Windows, we recommend Microsoft Expression Web 4, which is a free download (click here to get it).  The instructions that follow were tested using that program, but they all pretty much work the same.

COPY the athleteBio directory to a safe location, and work there

Locate the athleteBio directory in your owlcms installation. If you are running the Windows install, locate the "Competition Management" folder. There will be a shortcut to the athleteBio directory.

Repeat the following steps for all the athletes

This will take a while.  Don't say we did not warn you that showing pictures was a cute but time-consuming idea.

1- Copy the picture for the athlete

  • Get a membership or registration number for the athlete.  If you are hosting a national championship and each state or province has a code, then prefix the code.
    • In the athleteBio provided, you will find two files called qc4123.  This is me as an example, where I am from Québec (qc) and have membership number 4123.
    • You will be entering this membership number on the "Registration" page in the program (leftmost cell on the page) -- This is how the program will locate the athlete's file.
  • Get a suitably sized picture for the athlete.  You need a passport picture shape, or a square.  If the file is very large, use a program to resize it to something more sensible.  You want 400 pixels wide ideally.
  • Rename the picture using the membership number.  In my example, I renamed my picture qc4123.jpg
  • Add the file to the athleteBio folder.
  • PRO TIP: if taking pictures of a lot of athletes, do it "mug shot style", with each athlete holding his or her registration number.  Then crop the pictures :-) after renaming the files.

2- Create the bio file and link it to the proper picture

  • Start your HTML editor, open the file called "sampleAthlete.html"
  • Save the file under the membership as entered in the owlcms program: in our example, I saved my file as qc4123.html
  • Right-Click on the picture in the file , and select "Picture Properties". Change the file name to the proper membership name (e.g. qc4123.jpg).
  • Edit the information as you see fit.  Note that you don't need to provide the name on the page the owlcms program will do that, as well as the current weight, etc.

Copy back all your files to the official athleteBio directory

You're all set to go.  You can add files as you wish, or even fix files during the competition.  The files are looked-up anew every time they are shown.
